🌐 Mini Web Projects
A collection of fun and interactive mini front-end projects built using HTML, CSS, and JavaScript. These projects demonstrate core web development concepts and offer engaging UI features.

🚀 Projects Included
🗓️ Mini Calendar
A dynamic calendar interface that allows users to navigate through months and view dates interactively.

🌦️ Weather App
Fetches real-time weather data using OpenWeatherMap API. Displays city-based weather info with temperature, conditions, and icons.

❤️ Heart Animation
A visually appealing animated heart using CSS and JS. Demonstrates animation timing and keyframe usage.

🔘 Button Ripple Effect
A stylish button that creates a ripple animation on click, mimicking material design feedback.

🛠️ Technologies Used
HTML5

CSS3 (Animations, Flex/Grid)

JavaScript (DOM Manipulation, API Fetching)

📁 Project Setup
Simply clone the repository and open the desired project folder in your browser.
